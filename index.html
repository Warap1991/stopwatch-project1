STOPWATCH PROJECT

A stopwatch is a timekeeping device used to measure the elapsed time between a start event and a stop event or to measure the duration of a specific activity or process. It provides a simple and precise way to track time intervals. Here's a breakdown of the functions of a stopwatch:

    Start:The primary function of a stopwatch is to begin counting time when you press the "start" button. This initiates the measurement of elapsed time from zero.

    Stop:You can press the "stop" button to halt the stopwatch's timekeeping function. This allows you to freeze the timer at the current elapsed time.

    Reset:The "reset" button is used to reset the stopwatch back to zero. This is typically done after stopping the timer and before starting a new timing session.
